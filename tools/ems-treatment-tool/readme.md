# EMS Behandelingstool

## Werking
Deze tool helpt EMS-medewerkers bij het bepalen van de juiste behandeling.

## Gebruik
1. Vul patiëntgegevens in
2. Selecteer letsels per lichaamsdeel
3. Klik op analyseer

## Aanpassen
Alle instellingen zitten in config.json:
- wondtypes
- pijnlevels
- bloedverlies
- lichaamsdelen
